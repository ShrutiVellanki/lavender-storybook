## Themes

### ☀️ Lavender Dawn (Light)

A warm, soft palette inspired by Rosé Pine Dawn. Cream backgrounds, muted purple foregrounds, and gentle borders.

### 🌙 Lavender Moon (Dark)

A deep, moody palette inspired by Rosé Pine Moon. Dark purple backgrounds with soft lavender accents.

Switching is controlled by the `dark` class on the document root:

```html
<html>              <!-- Lavender Dawn -->
<html class="dark"> <!-- Lavender Moon -->
```

## Quick Setup

### 1. Install dependencies

```bash
npm install clsx tailwind-merge lucide-react tailwindcss-animate
npm install recharts          # only if using Chart components
```

| Package | Purpose |
|---------|---------|
| `clsx` + `tailwind-merge` | The `cn()` class-merging utility used by every component |
| `lucide-react` | Icons (chevrons, close, search, spinner, etc.) |
| `tailwindcss-animate` | Accordion expand/collapse and spinner animations |
| `recharts` | Only needed for Chart components |

### 2. Add the `cn()` utility

Create `lib/utils.ts`:

```ts
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

### 4. Configure Tailwind

Map the CSS variables to Tailwind utilities in your `tailwind.config.ts`:

```ts
import type { Config } from "tailwindcss"

const config: Config = {
  darkMode: ["class"],
  content: ["./src/**/*.{js,ts,jsx,tsx}"],
  theme: {
    extend: {
      colors: {
        background: "var(--background)",
        foreground: "var(--foreground)",
        card: { DEFAULT: "var(--card)", foreground: "var(--card-foreground)" },
        popover: { DEFAULT: "var(--popover)", foreground: "var(--popover-foreground)" },
        primary: { DEFAULT: "var(--primary)", foreground: "var(--primary-foreground)" },
        secondary: { DEFAULT: "var(--secondary)", foreground: "var(--secondary-foreground)" },
        muted: { DEFAULT: "var(--muted)", foreground: "var(--muted-foreground)" },
        accent: { DEFAULT: "var(--accent)", foreground: "var(--accent-foreground)" },
        destructive: { DEFAULT: "var(--destructive)", foreground: "var(--destructive-foreground)" },
        border: "var(--border)",
        input: "var(--input)",
        ring: "var(--ring)",
      },
      borderRadius: {
        lg: "var(--radius)",
        md: "calc(var(--radius) - 2px)",
        sm: "calc(var(--radius) - 4px)",
      },
    },
  },
  plugins: [require("tailwindcss-animate")],
}
export default config
```

### 5. Use ThemeProvider

Wrap your app to enable theme switching with localStorage persistence:

```tsx
import { ThemeProvider } from "@/components/ui/ThemeProvider"

function App() {
  return (
    <ThemeProvider defaultTheme="dark" storageKey="lavender-theme">
      {/* your app */}
    </ThemeProvider>
  )
}
```

Toggle programmatically with the `useTheme` hook:

```tsx
import { useTheme } from "@/components/ui/ThemeProvider"

const { theme, setTheme } = useTheme()
setTheme(theme === "light" ? "dark" : "light")
```

## Spacing and Radius

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| `--radius` | `0.625rem` | `rounded-lg` | Cards, buttons, inputs |
| | `calc(var(--radius) - 2px)` | `rounded-md` | Inner elements |
| | `calc(var(--radius) - 4px)` | `rounded-sm` | Badges, pills |
